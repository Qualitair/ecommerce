=== WooCommerce Amazon FPS ===
Author: skyverge
Tags: woocommerce
Requires at least: 3.8
Tested up to: 4.1
Requires WooCommerce at least: 2.1
Tested WooCommerce up to: 2.3

Adds the Amazon FPS Payment Gateway to your WooCommerce store, allowing customers to securely use their Amazon account with single purchases, pre-orders, subscriptions, and more!

See http://docs.woothemes.com/document/amazon-fps/ for full documentation.

== Installation ==

1. Upload the entire 'woocommerce-gateway-amazon-fps' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
